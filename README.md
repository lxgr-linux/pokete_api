# Pokete API

An API to get (Pokete, Attack etc.) data from [Pokete](https://github.com/lxgr-linux/pokete).

## Usage
To lauch the server:
```shell
./run.sh
```

To not download data files:
```shell
./run.sh --no-download
```

To just update the data files without running the server:
```shell
./run.sh --update
```

To show usage information
```shell
./run.sh --help
```
